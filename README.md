# Monte Carlo Option Pricer

이 저장소에는 몬테카를로 시뮬레이션으로 옵션 가격을 추정하는 간단한 파이썬 스크립트가 들어 있습니다. 사용자가 직접 페이오프 함수를 문자열로 입력하여 다양한 파생상품 구조를 실험할 수 있습니다.

## 사용법

```bash
python monte_carlo_option.py \
    --s0 100 \
    --r 0.03 \
    --sigma 0.2 \
    --t 1 \
    --paths 20000 \
    --steps 50 \
    --k 110 \
    --payoff "max(ST - K, 0)" \
    --seed 123
```

### 주요 옵션
- `--s0`: 기초자산의 초기 가격.
- `--r`: 무위험이자율(연율).
- `--sigma`: 변동성(연율).
- `--t`: 만기까지의 기간(연 단위).
- `--steps`: 경로당 시뮬레이션 단계 수. 최소 1.
- `--paths`: 몬테카를로 경로 수.
- `--k`: 스트라이크 (페이오프에서 사용할 선택적 값).
- `--payoff`: `ST`(만기 가격)와 `S0`, `K`, `r`, `sigma`, `T`, `np` 등을 사용할 수 있는 파이썬 표현식.
- `--seed`: 난수 시드.

출력은 할인된 옵션 가격 추정치와 표준 오차입니다.
